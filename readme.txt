	Email notification for Question Answer

== Description ==

Get email notification when any action occurred via Question Answer plugin.

### Question Answer by http://pickplugins.com
* [Get Question Answer !&raquo; ](https://wordpress.org/plugins/question-answer/)
* [Live Demo !&raquo; ](http://www.pickplugins.com/demo/question-answer/?ref=wordpress.org)
* [Documentation !&raquo; ](http://pickplugins.com/documentation/question-answer/?ref=wordpress.org)


<strong>Plugin Features</strong>

Email notification for - 

* New Question Submitted
* New Question Published
* New Answer Submitted
* New Answer Published
* Question solved
* Question unsolved
* Comment flag
* Comment unflag
* Answer Voted Up
* Answer Voted Down






<strong>Translation</strong>

Pluign is translation ready , please find the 'en.po' for default translation file under 'languages' folder and add your own translation. you can also contribute in translation, please contact us http://www.pickplugins.com/contact/



== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Question Answer</strong>" activate it.<br />
